## AI-Based Airline Complaint Severity Detection & Service Risk Classification System

This project focuses on building an end-to-end Natural Language Processing (NLP) system that analyzes airline customer reviews and automatically classifies complaint severity and service risk. The system performs text preprocessing, sentiment analysis, severity classification, risk detection, exploratory data analysis, and model comparison across Machine Learning, Deep Learning, and Transformer architectures.

The project implements multiple models including Logistic Regression, Support Vector Machine, Naive Bayes, Random Forest, BiLSTM neural networks, and DistilBERT transformer models. The models are evaluated using Accuracy, Precision, Recall, and Macro F1-score, and the best-performing models are selected for deployment.

A multi-page Streamlit application is developed to provide an interactive dashboard where users can:

* View project overview and dataset information
* Explore EDA visualizations and complaint trends
* Predict complaint severity and service risk for single reviews
* Upload CSV files for bulk predictions
* Download prediction results

This project demonstrates end-to-end NLP pipeline development, model comparison, explainable risk scoring, and deployment of machine learning models using Streamlit.
